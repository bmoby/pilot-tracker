# Активные планы

В этой папке хранятся планы разработки, по которым работа еще не завершена.

Каждый новый план разработки сначала создается здесь.

Новый план создается по шаблону `../шаблон-плана.md` после создания отдельной ветки плана.

Файл плана остается в этой папке до завершения разработки и проверки владельцем проекта.
