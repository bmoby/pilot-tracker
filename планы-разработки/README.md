# Планы разработки

Эта папка хранит рабочие планы разработки Pilot Tracker.

Текущие планы находятся в `активные-планы/`.

Завершенные планы находятся в `завершенные-планы/`.

Шаблон нового плана хранится в `шаблон-плана.md`.

Когда владелец проекта просит составить план, агент должен сначала обновить `main`, создать ветку плана по формату `plan/YYYY-MM-DD-korotkoe-nazvanie`, а затем создать файл плана в `активные-планы/` по шаблону.

План считается активным с момента создания до момента, когда владелец проекта завершил тестирование и попросил подготовить изменения к отправке в GitHub.

Перед commit и push файл завершенного плана нужно переместить из `активные-планы/` в `завершенные-планы/`, чтобы сохранить историю выполненной работы.
