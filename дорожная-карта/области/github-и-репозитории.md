# GitHub и репозитории

Этот файл хранит хронологию завершенных работ, которые относятся к GitHub и локальным репозиториям студентов.

Сюда относятся клонирование, обновление через Git, работа с веткой `main`, GitHub CLI, локальные копии репозиториев, review-копии, открытие кода в VS Code и ограничения read-only для студенческих проектов.

## Формат записи

```text
## ГГГГ-ММ-ДД - Название завершенного плана

- План: `планы-разработки/завершенные-планы/<имя-файла>.md`
- Сделано: что изменилось в работе с GitHub и репозиториями
- Влияние: как это влияет на обновление и проверку проектов
- Проверка: как была проверена работа
```

## Записи

## 2026-05-10 - Обновление проектов и история

- План: `планы-разработки/завершенные-планы/обновление-проектов-и-история.md`
- Сделано: добавлен Git-адаптер для клонирования новых репозиториев, чтения `HEAD`, проверки чистоты рабочей копии, получения `origin/main`, подсчета новых коммитов и приведения чистой основной копии к `origin/main`.
- Влияние: преподаватель может запускать обновление проектов, а приложение сохраняет результат без `commit`, `push`, `merge` и изменения студенческих репозиториев от имени приложения.
- Проверка: автоматические тесты покрыли первую загрузку, повторное обновление с новыми коммитами и ошибку одного проекта без остановки общего запуска; выполнен `pnpm check`.

## 2026-05-14 - Review-копии и открытие кода в VS Code

- План: `планы-разработки/завершенные-планы/review-копии-и-vscode.md`
- Сделано: добавлены операции Git worktree для создания, проверки, переиспользования и безопасного пересоздания чистых review-копий выбранных обновлений.
- Влияние: преподаватель открывает код конкретного обновления на сохраненном коммите, а основная локальная копия студенческого проекта остается на `main` для будущих обновлений.
- Проверка: автоматические тесты покрыли создание review-копии, повторное открытие, грязную копию, отсутствующий коммит и ошибку открытия VS Code; выполнен `pnpm check`.

## 2026-05-14 - Диагностика локальных инструментов

- План: `планы-разработки/завершенные-планы/диагностика-локальных-инструментов.md`
- Сделано: добавлена проверка доступности `git --version`, `gh --version`, `gh auth status` и `code --version` из настроек локальных инструментов.
