# ИИ-анализ

Этот файл хранит хронологию завершенных работ, которые относятся к ИИ-анализу в Pilot Tracker.

Сюда относятся запуск локального ИИ-анализа, подготовка входных данных, ИИ-рапорты, черновики комментариев, связь анализа с обновлениями проекта и проверяемость результата ИИ.

## Формат записи

```text
## ГГГГ-ММ-ДД - Название завершенного плана

- План: `планы-разработки/завершенные-планы/<имя-файла>.md`
- Сделано: что изменилось в ИИ-анализе
- Влияние: как это помогает проверке проектов
- Проверка: как была проверена работа
```
